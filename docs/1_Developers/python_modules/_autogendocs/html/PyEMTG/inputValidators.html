inputValidators module¶

inputValidators.py¶

This file contains various classes and functions to support input validation for wx.TextCtrl inputs. It includes NumberRangeValidator, FilePathValidator, ListFormatValidator, and AlphaNumericFileValidator.

checkBounds(txt, again=True)[source]¶

Checks whether the given text is a valid number, and whether that number is within the range

  • If the value is not a valid number, but could be with additional input the return value is (True, None)

  • If the value contains characters that do not form a valid number the return value is (False, None)

  • If the value is a valid number but is not in the range the return value is (False, value)

  • If the value is a valid number and is in the range the return value is (True, value)

Parameters:

  • txt (string) – The string to be checked against the range

  • again (bool, defaults to True) – Whether the string should be checked again with the last character removed, useful for checking a value like “1e”

Return type:

tuple(bool,float).

inputValidators.rereplace(text, toReplace, replacer)[source]¶

Repeats replace until there are no instances of the toReplace string (e.g. if text=C:\\\\Users, toReplace=\\, replacer=\ then C:\\\\Users -> C:\\\Users -> C:\\Users -> C:\Users)

Parameters:

  • text (string) – The string that should be modified

  • toReplace (string) – The string to replace in “text”

  • replacer (string) – The string to replace “toReplace” with

Return type:
