combine_PEATSA_output_csv_files module¶

combine_PEATSA_output_csv_files.py¶

Contains function that combines the contents of multiple PEATSA output csv files (the Iteration#.csv files).

It only makes sense to use this function if all the input .csv files have the same columns.

combine_PEATSA_output_csv_files.combine_PEATSA_output_csv_files(inputFiles, outputFile)[source]¶

Function that combines the contents of multiple PEATSA output csv files (the Iteration#.csv files).

It only makes sense to use this function if all the input .csv files have the same columns.

Parameters:

  • inputFiles (List of strings) – List of full file paths to all input files to be combined. Column headers for the output file are taken from the first element in the list

  • outputFile (String) – Full file path to the csv file to which the combined contents are to be written.

Return type:

None.
