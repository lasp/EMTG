tooltip_loader.load_tooltips(panel, items)[source]¶

Loads the tooltips for a set of items in a panel

Parameters:

  • panel (string) – The name of the panel that the inputs belong to. Corresponds to a folder in docs/shared_latex_inputs/tooltips/. Valid values are “global”, “spacecraft”, “journey”, “solver”, “physics”, and “output”.

  • items (list[tuple(2)]) – A list of two element tuples, the first of which is the name of the text file the tooltip should be loaded from, excluding the file extension (assumed to be *.txt), and the second of which is the widget to attach the tooltip to.

Return type:

None.
