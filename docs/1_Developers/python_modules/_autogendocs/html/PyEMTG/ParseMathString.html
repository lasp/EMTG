ParseMathString module¶

ParseMathString.ParseMathString(math_string)[source]¶

Function takes in a string (e.g. “40+2”, “3600*4”, etc.), converts the string into the mathematical operation it describes, and returns the result. Current constraints/assumptions:

  1. Only 1 operation can be performed (i.e. can only process a single addition, multiplication, exponent, etc. at a time). Basic operations like “3600*4” or “121^0.5” are acceptable; computations

    with additional steps/operations like “60*60*4” or “121^0.5+1” will throw an error.

  2. All numbers are assumed to be floats.

Parameters:

math_string (String) – String containing a mathematical expression to be computed.

Returns:

result – The first entry in the tuple is a pass/fail flag. True means a numerical value was found; provided in the second entry of the tuple. False means a numerical value could not be found, and the second element of the tuple contains an error message.

Return type:

Tuple
